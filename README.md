## 🪄 프로젝트 실행 방법
1. git clone하여 프로젝트를 내려받습니다.
    ```bash
    git clone https://github.com/spdhsrnvl123/Coin.git
    ```
2. 패키지를 설치합니다.
    ```bash
    npm install
    ```
3. 프로젝트를 실행합니다.
    ```bash
    npm start
    ```
